# Doordeer Smart Intercom — HA Add-on Repository

[![Add to Home Assistant](https://my.home-assistant.io/badges/supervisor_add_addon_repository.svg)](https://my.home-assistant.io/redirect/supervisor_add_addon_repository/?repository_url=https://github.com/Beef-well/doordeer)

## Installation

1. Click the badge above **or** go to **Settings → Add-ons → Add-on Store → ⋮ → Repositories**
2. Add: `https://github.com/Beef-well/doordeer`
3. Install **Doordeer Smart Intercom**
4. Configure your device IP and RC4 key, then start the addon

## Add-ons in this repository

### Doordeer Smart Intercom
Full LAN integration for doordeer doorbell/video intercom devices.
Supports door unlock, RTSP camera stream, JPEG snapshots, access logging, and credential management.
